### 다중 분기
- `if`, `elif`, `else` 문
- 먼저 `참`이 되는 조건문을 실행
- 조건문이 실행되면 블록이 종료
```
x = 0
if x < 2 :
    print('small')
elif x < 20  :
    print('Medium')
else :
    print('LARGE')              Output : small
print('All done')                        All done
```
